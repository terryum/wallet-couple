## 1. 지출 카테고리 (Expense Categories)

### Set A: AI 자동 분류 대상
AI가 이용처명을 분석하여 자동으로 분류하는 카테고리입니다.

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| 식료품 | 마트, 슈퍼, 편의점에서 구매한 식재료, 생필품 | 이마트, GS25, 하나로마트 |
| 외식/커피 | 음식점, 카페, 배달음식, 커피, 음료 | 스타벅스, 맥도날드, 배달의민족 |
| 쇼핑 | 의류, 잡화, 온라인쇼핑, 백화점 | 쿠팡, 무신사, 현대백화점 |
| 관리비 | 아파트 관리비, 공과금, 가스비, 전기요금 | 관리비, 한국전력, 서울도시가스 |
| 통신/교통 | 휴대폰요금, 인터넷, 교통비, 주유, 택시, KTX | SK텔레콤, 코레일, GS칼텍스 |
| 육아 | 어린이집, 유치원, 학원, 아이용품, 장난감, 아이 병원비 | 키즈카페, 토이저러스, 소아과 |
| 병원/미용 | 병원, 약국, 의료비, 미용실, 피부관리, 네일, 화장품 | 올리브영, 이비인후과, 헤어샵 |
| 기존할부 | 할부 결제 (자동차, 가전제품 등 기존 할부금) | 자동차할부, 가전할부 |
| 대출이자 | 대출이자, 카드이자, 금융비용 | 주택담보대출이자, 신용대출이자, 토스뱅크이자 |
| 양육비 | 양육비 이체, 양육 관련 고정 지출 | 양육비 |
| 세금 | 국세, 지방세, 자동차세, 재산세, 주민세 | 경찰청, 인천서구, 68저, 179호 |

### Set B: 사용자 수동 변경 전용
AI가 자동 분류하지 않으며, 사용자가 직접 변경해야 하는 카테고리입니다.

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| 여행 | 여행 관련 숙박, 항공, 관광 | 호텔, 에어비앤비, 대한항공 |
| 부모님 | 부모님 용돈, 부모님 관련 지출 | 부모님용돈 |
| 친구/동료 | 친구나 동료와의 모임, 회식 | 회식, 동창회 |
| 경조사/선물 | 축의금, 조의금, 선물 구입 | 결혼식, 장례식, 생일선물 |
| 가전/가구 | 가전제품, 가구 구입 | 삼성전자, 이케아, LG전자 |
| 기타 | 위 카테고리에 속하지 않는 기타 지출 | - |

## 2. 소득 카테고리 (Income Categories)

소득은 모두 AI가 자동 분류합니다.

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| 급여 | 월급, 정기 급여, 회사에서 받는 급여 | 월급여, 코스맥스(주) |
| 상여 | 보너스, 인센티브, 추석/명절 상여금 | 상여금, 명절보너스 |
| 정부/환급 | 정부 지원금, 세금 환급, 연말정산 환급, 육아수당 | 국세환급, 아동수당, 연말정산환급 |
| 강연/도서 | 강연료, 원고료, 인세, 컨설팅비, 부업 수입 | 강연료, 원고료 |
| 금융소득 | 이자, 배당금, 투자 수익, 예금 이자 | 예금이자, 배당금 |
| 기타소득 | 중고 판매, 경품, 기타 수입 | 당근마켓, 경품 |

## 3. 카테고리 수정 가이드

### 새 카테고리 추가 시 수정해야 할 파일들:

1. **이 파일 (CATEGORIES.md)** - 카테고리 문서 업데이트
2. **`src/types/index.ts`** - 타입 정의 배열에 추가
   - 지출: `EXPENSE_CATEGORY_SET_A` 또는 `EXPENSE_CATEGORY_SET_B`
   - 소득: `INCOME_CATEGORIES`
3. **`src/lib/classifier/index.ts`** - AI 분류 설명 추가
   - 지출: `CATEGORY_DESCRIPTIONS`
   - 소득: `INCOME_CATEGORY_DESCRIPTIONS`
4. **`src/components/transactions/TransactionRow.tsx`** - 배지 색상 추가
   - `CATEGORY_COLORS` 객체에 새 카테고리 색상 정의

### 카테고리 색상 가이드:
- **지출 카테고리**: 각 카테고리에 맞는 의미있는 색상 사용
- **소득 카테고리**: 녹색 계열 (`green`, `lime`, `teal`, `cyan`) 사용
