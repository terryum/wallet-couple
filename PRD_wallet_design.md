# Design System (Apple/Clean Style)

## 1. Color Palette (Tailwind)
- Background: `bg-slate-50` (Very light grey for page)
- Surface (Card): `bg-white` (Pure white for cards)
- Primary Brand: `#FF4D88` (Cosmax Pink or K-Beauty Point Color)
- Text Main: `text-slate-900` (Almost Black)
- Text Sub: `text-slate-500` (Grey for descriptions)
- Border: `border-slate-100` (Very subtle)

## 2. Typography
- Font Family: "Pretendard" or "Inter", sans-serif
- Headings: Bold (700), Tight tracking (`tracking-tight`)
- Body: Regular (400), Relaxed line-height
- Size: Mobile-first. Base size 15px.

## 3. Component Style (Shadcn/UI based)
- **Cards:** White background, `rounded-2xl` (Larger radius), `shadow-sm` (Subtle shadow).
- **Buttons:** Full width on mobile, `h-12` (Tall accessible buttons), `rounded-xl`.
- **Charts:** Clean lines, Minimal grid lines, use Brand Color for main data.

## 4. Spacing (Mobile Optimized)
- Page Padding: `px-5` (Consistent horizontal padding)
- Section Gap: `gap-6`
- Card Padding: `p-5`
